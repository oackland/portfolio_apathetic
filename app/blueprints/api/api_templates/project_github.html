API Documentation

Overview

Welcome to our API documentation. This document provides an overview of available endpoints and how to use them.

Endpoints

GET /api/items

Returns a list of all items.

Request:
GET /api/items

Response:
[
    {
        "id": 1,
        "name": "Item 1"
    },
    {
        "id": 2,
        "name": "Item 2"
    }
]

POST /api/items

Creates a new item.

Request:
POST /api/items
Content-Type: application/json

{
    "name": "New Item"
}

Response:
{
    "id": 3,
    "name": "New Item"
}

Errors

If there's an error with your request, the API will return a response with an appropriate error code and a message describing the problem.
